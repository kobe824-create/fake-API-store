# Fake API Store

Written by @iMarand

This project is a web application that simulates an online store using the FakeStore API. It allows users to browse products, add them to a cart, and checkout. Built with Flask.

## Prerequisites

- Python 3.x

## Installation

1. Clone the repository.

2. Install dependencies:

   ```
   pip install flask requests
   ```

## Running the Project

1. Run the application:

   ```
   python app.py
   ```

2. Open your browser and go to http://localhost:5000
